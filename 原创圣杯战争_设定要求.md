# 原创圣杯战争——设定需求文档

---

## 一、核心框架

### 1.1 基本结构

七个参战者，每人独立参战。不存在"御主"和"从者"的区分。每个参战者本身就是拥有英灵力量的战斗单位。

### 1.2 英灵力量的承载方式

七个参战者身上承载英灵力量的方式不必统一，每个人可以不同。以下为可选方式（非穷举，可混用）：

| 承载方式 | 描述 | 特征 |
|---------|------|------|
| **完全附体** | 英灵的灵魂完全占据参战者的肉体，参战者本人的意识被压制或共存 | 战斗力最接近原始英灵，但肉体承载有极限，可能有时间限制或反噬 |
| **能力继承** | 参战者本人保留完整意识，但获得了英灵的部分或全部技能/宝具 | 使用英灵能力需要消耗自身资源（魔力、生命力等），上限低于原始英灵 |
| **血脉觉醒** | 参战者是英灵的后裔或转世，血脉中沉睡的力量被圣杯唤醒 | 力量与自身同源，排斥反应最小，但需要觉醒过程，初期可能不完全 |
| **概念寄生** | 英灵不以灵魂形式存在，而是以"概念"形式寄生于参战者的某件物品或身体部位上 | 力量集中于特定媒介（武器、器官、纹身等），失去媒介即失去力量 |
| **契约融合** | 参战者与英灵签订契约，两者意识共存，可在人格之间切换 | 最灵活，但人格切换有成本（精神负担、记忆冲突），两个意志可能产生分歧 |
| **其他** | 不限于以上分类，只要自洽即可 | — |

**每个参战者的承载方式必须影响其战斗风格和叙事弧线。** 承载方式不是标签——它决定了这个人在战争中的行为逻辑、极限、弱点和成长方向。

### 1.3 职阶的处理

职阶（Saber/Archer/Lancer/Rider/Caster/Assassin/Berserker）保留为分类标签，但不是"从者的固有属性"——而是"英灵力量在参战者身上的表现倾向"。

一个继承了骑士型英灵能力的参战者，战斗风格自然偏向Saber类。但因为参战者本人有自己的性格和判断，实际行为可能偏离职阶预期——比如承载了Berserker型英灵力量的人，如果本人极度理性，可能用理性的方式驾驭狂化能力而非失控暴走。

**职阶是参考，不是枷锁。**

### 1.4 退场机制

参战者被消灭即物理死亡。本设定中不存在英灵座，英灵的意识从历史信息中被抽取而来，参战者死亡意味着英灵意识的完全消散，不可逆。

## 二、总体基调

### 2.1 核心基调

- **严肃向**：为了实现愿望的大战。每个参战者都有不得不战的理由，每次战斗都有不可挽回的代价。
- **热血向**：战斗荡气回肠。热血的来源是选择的重量、极限的触碰、对手之间的尊重。
- **死亡真实**：没有主角光环。选择决定存亡。错误的判断、鲁莽的决定、实力不足都可能导致死亡。
- **情报迷雾**：正文中不直接标注英灵真名与宝具名。所有信息必须通过战斗观察、对话暗示、线索推理逐步揭露。

### 2.2 热血的定义

热血不等于"一直在打架"。热血来自：
- **选择的重量**——明知代价仍然做出决定
- **极限的触碰**——宝具/大招是最后手段而非开局技能
- **对手的尊重**——胜者不嘲讽，败者不窝囊

禁止"无意义的华丽战斗"——每一场战斗必须对局势产生实质性影响。

### 2.3 节奏控制

以事件里程碑定义阶段，而非百分比：

| 阶段 | 定义 | 叙事重心 | 消灭人数 |
|------|------|---------|---------|
| **序章阶段** | 战争开始 → 第一次死亡 | 日常展示、角色建立、试探性接触、情报收集。小规模交锋可以有，但致命战斗应在本阶段末期。 | 1 |
| **中盘阶段** | 第一次死亡 → 第四次死亡 | 最密集最复杂的阶段。临时同盟的建立与瓦解，多方博弈，连锁战斗。 | 2~3 |
| **终盘阶段** | 剩余三人及以下 | 情报迷雾散去，隐藏设定揭露，角色被迫面对真相。紧迫感上升。 | 1~2 |
| **最终决战** | 剩余两人 | —— | 决出胜者 |

这个分布不是铁律，但偏离需要充分的叙事理由。

### 2.4 视角权重

- 主角视角占比不超过55%（以场景数为计量单位——"场景"定义为连续的、以同一视角人物为焦点的叙事段落）。
- 至少三个非主角参战者需要有完整的内部戏份（日常、对话、决策过程），这样他们被消灭时读者才会有感觉。
- 至少一个参战者的死亡应从**被消灭者的视角**来写。

## 三、角色设计原则

### 3.1 基础要求

每个参战者必须具备：
- 完整的个人身份（国籍、职业、日常生活）
- 英灵力量的承载方式
- 英灵来源（真名、传说来源、文化圈）
- 详细的能力设定（六维参数、技能、宝具）
- 明确的愿望/目标
- 不同质化的战斗风格

### 3.2 "活在战争之外"测试

每个参战者必须通过测试：**如果没有圣杯战争，这个人在做什么？**

答案不能是"等待圣杯战争"。必须有独立于战争的日常生活、职业、人际关系。这些日常元素必须在战争中被具体地影响。

### 3.3 参数规则

- 能力值等级：**E、D、C、B、A、S、EX**
- 数据化：E=10，D=20，C=30，B=40，A=50，S=100，EX=论外（>100）
- **禁止使用"+"标记**
- A到S的跳跃不是量的递增而是质的变化——S级意味着该维度上已达非人领域，对资源消耗的影响也远超A级
- 七个参战者的六维参数总值应有预算区间，防止出现六维全A的超人。建议：单人六维总值区间为120~240（即平均D~A之间）

### 3.4 参战者多样性

#### 3.4.1 英灵文化圈多样性

文化圈采用大区划分：东亚 / 南亚 / 中东-北非 / 欧洲 / 美洲 / 撒哈拉以南非洲 / 其他（大洋洲、中亚等）

- 每个大区最多两个英灵
- 七个英灵至少覆盖**四个不同大区**
- 不能出现Fate系列世界书中已有的英灵人物，**必须是原创英灵**
- 可以使用历史上真实存在但不那么有名的人物，也可以使用民间传说中没有确定身份的角色

#### 3.4.2 承载方式多样性

七个参战者的英灵承载方式**至少覆盖三种不同类型**。

#### 3.4.3 战斗能力光谱

七人的战斗能力必须覆盖以下光谱：
- 至少一人是**纯近战型**（依赖身体素质和武器技巧）
- 至少一人是**纯远程/支援型**（依赖魔术、计算、信息操控等）
- 至少一人是**混合型**（远近皆能但都不是最强）
- 至少一人的**战斗力在初始状态下明显弱于其他人**（但有成长潜力或特殊优势）
- 至少一人的**战斗力在初始状态下明显强于其他人**（但有明确的硬限制或代价）

#### 3.4.4 愿望多样性

- 七人的愿望/目标不可超过两人相同类型
- 至少一人的愿望与圣杯战争本身有关（如摧毁圣杯、改变规则等）
- 至少一人的愿望是极度个人化的（与世界大局无关，只关乎自己的某个执念）
- 至少一人在战争过程中愿望发生改变

### 3.5 强度平衡

#### 3.5.1 克制关系

七人之间必须形成克制网络（不必是严格环形，可以是更复杂的网状结构）。

每一条克制关系都必须有：
- **具体的原因**（能力相克、属性压制、战术天然优势等）
- **环境依赖**（灵脉条件、时间段、地形、一对一vs多方混战等情境因素）
- **破解条件**（没有绝对克制）

克制关系表格式：

| A → B | 克制原因 | 克制强度（强/中/弱） | 环境依赖 | 破解条件 |
|-------|---------|-------------------|---------|---------|
| （每条克制关系一行） | | | | |

#### 3.5.2 宝具经济表

| 参战者 | 宝具名（内部代号） | 等级 | 类型 | 每次消耗 | 冷却时间 | 副作用 | 战争最大使用次数 | 备注 |
|-------|-----------------|------|------|---------|---------|--------|----------------|------|
| （每个宝具一行） | | | | | | | | |

### 3.6 英灵力量对参战者的影响

这是最重要的角色设计维度——英灵的力量和参战者本人之间的张力。

影响必须覆盖多个维度，七人的影响类型**至少覆盖以下三个维度中的两个**：

**精神维度：**
- 英灵意识/记忆的侵蚀
- 使用宝具时闪回英灵记忆产生精神冲击
- 英灵性格在激烈战斗中渗透到参战者行为中
- 过度使用导致自我认知模糊
- 英灵的执念/未竟之愿干扰参战者判断

**肉体维度：**
- 使用英灵力量后身体部位不可逆变异
- 感官扭曲（视觉色偏、听觉频段偏移、痛觉阈值异常等）
- 寿命缩短
- 身体逐渐向英灵生前的体质/外貌靠拢

**社会维度：**
- 行为模式突变导致身边人察觉异常
- 灵脉波动引发周围普通人的不适反应
- 无法维持战前的日常身份和人际关系
- 英灵时代的习惯/价值观在日常中不自觉流露

具体形式因人而异，但**不可为零**。

**每个参战者的设定中必须包含"英灵力量对本人的影响"条目，明确标注涉及哪个维度。**

## 四、世界观构建原则

### 4.1 圣杯系统

#### 4.1.1 基础框架

- 独立于已有作品的自洽体系
- 核心机制有明确的历史渊源（谁建造的、为什么建造、中途经历了什么）
- 圣杯的真实状态必须在设定文档中写死答案，不可留"待定"
- 不要让圣杯完全失去实现愿望的机能，也不要让圣杯通过过于扭曲的方式实现愿望

#### 4.1.2 参战资格机制

圣杯直接选定七个参战者。必须明确：
- 选定的标识是什么？（令咒？印记？其他？）
- 选定的标准是什么？（灵脉亲和度？愿望强度？英灵力量适配性？随机？）
- 选定后参战者是否可以拒绝？拒绝的后果是什么？

#### 4.1.3 胜利条件

- 适配单人制的胜利条件
- 是否仍然是"最后一人站着"？还是有其他条件？
- 节点机制是否保留？如果保留，节点在何处？

#### 4.1.4 系统的"呼吸节律"

圣杯系统必须有一个可感知的周期性变化（潮汐、日夜、季节等），直接影响战场条件。

这个节律必须是公开信息——关键不在于谁知道，而在于谁能利用。

#### 4.1.5 所有悬疑必须有预设答案

标注为"悬疑要素"的设定，在设定文档中必须写死答案。正文中可以不揭露，但设定文档中必须有。**禁止"真相未知"开局。**

### 4.2 地理与战场

#### 4.2.1 基础要求

- 基于真实地理构建战场，地点真实存在
- 每个区域应有独特的战术价值（灵脉强度、地形优势、人口密度等）
- 设置非正式的缓冲区/中立区

#### 4.2.2 移动时间表

| 起点→终点 | 参战者高速移动 | 普通人交通 | 特殊移动方式 |
|----------|--------------|----------|------------|
| （关键路线一行） | | | |

#### 4.2.3 人口密度约束

每个区域的人口密度（白天/夜晚）必须标注为高/中/低三级。人口密度直接约束战斗烈度：

- **高密度区**：大规模战斗和对军级能力将造成大量平民伤亡，引发现实军事和警务力量的全面响应。参战者身份暴露风险极高。
- **中密度区**：可小规模交战，需掩护措施。过度破坏仍会引发现实响应。
- **低密度区**：不限制。

此外，人口密集区的高密度人类意识对灵脉有天然的噪声屏蔽效应，使得大规模魔术在密集区效果衰减。

#### 4.2.4 灵脉等级分布表

| 区域 | 灵脉等级 | 战术价值 | 白天人口密度 | 夜晚人口密度 |
|------|---------|---------|------------|------------|
| （每个关键区域一行） | | | | |

### 4.3 势力格局

#### 4.3.1 初始关系矩阵

七人之间的初始关系必须明确。矩阵为初始状态，非固定——必须为每对关系预设至少一个变化触发条件。

#### 4.3.2 阵营形成的逻辑

七人制下不存在固定阵营——但临时同盟、默契互不侵犯等关系可以存在。同盟必须有**明确的利益基础**和**明确的破裂条件**。

### 4.4 信息初始分布

每个参战者在战争开始时的已知信息必须明确定义。不同参战者的信息差本身是冲突来源。

**信息初始分布表：**

## 五、隐藏设定原则

### 5.1 隐藏设定的结构

所有隐藏设定采用**"真相+触发条件+揭露方式"**的三段结构。设定文档中必须写死每个隐藏真相的完整答案，并定义在什么事件发生后这个真相可以被揭露。

**隐藏设定表：**

| 隐藏真相 | 级别 | 设定答案（完整） | 触发条件 | 揭露方式 |
|---------|------|----------------|---------|---------|
| （每条一行） | A/B/C | | | |

### 5.2 级别定义

| 级别 | 定义 | 触发条件要求 |
|------|------|------------|
| A级（核心悬疑） | 关乎圣杯战争本质、胜利条件真相等 | 必须在终盘阶段或之后才可能触发。触发前至少有3处可回溯的异常现象已出现在叙事中。 |
| B级（角色秘密） | 关乎单个参战者的隐藏身份、真实愿望等 | 中盘阶段可触发。触发前至少有1处行为层面的不一致已出现。 |
| C级（角色内心） | 角色的内在动机和情感 | 随剧情自然展开，不需预设触发条件，但揭露时必须与此前行为一致。 |

### 5.3 规则

- A级隐藏设定的异常现象必须分布在至少两个不同角色的视角/经历中。
- 异常现象应是行为层面或环境层面的，不可是"角色直接说出关键信息但被打断"。
- 禁止埋设永远不会被揭露的伏笔。唯一例外：世界背景悬疑可以不在正文揭露，但设定文档中必须有完整答案。

## 六、能力经济

### 6.1 资源体系

每个参战者是自给自足的——但自给自足不意味着无限资源。

每个参战者的资源体系必须明确以下条目：

| 条目 | 描述 |
|------|------|
| 资源来源 | 自身魔术回路？英灵力量的被动产出？灵脉汲取？其他？ |
| 资源上限 | 总储备量（以标准化单位计） |
| 日常消耗 | 维持英灵力量的基础消耗 |
| 可用余量 | 扣除日常消耗后可用于战斗/宝具的余量 |
| 恢复速率 | 非战斗状态下每小时恢复量 |
| 赤字后果 | 资源耗尽后的后果（英灵力量失控？肉体损伤？意识消退？） |

### 6.2 标准化单位

定义一个标准化单位（例如"1单位=C级魔术一次的消耗"）。所有消耗量、储备量、恢复量都以此单位计量。

### 6.3 参战者资源表

| 参战者 | 资源来源 | 总储备 | 日常消耗/天 | 可用余量 | 恢复速率/时 | 赤字后果 |
|-------|---------|-------|-----------|---------|-----------|---------|
| （每人一行） | | | | | | |

### 6.4 战斗消耗不可回避

- 每场战斗后必须体现消耗后果
- 恢复时间必须具体（"48小时"而非"一段时间"）
- 两场战斗间隔不足以完全恢复时，第二场必须以削弱状态开始

## 七、叙事机制

### 7.1 同盟的有效期

- 同盟必须有明确的利益基础和明确的破裂条件
- 同盟的叙事价值在于信任的测量——双方都在试探底线
- 盟友必须在至少一场战斗中提供决定性支援（不是旁观），然后在后续因利益分歧面临选择

### 7.2 情报的价值链

- 获取情报需要付出代价
- 情报有时效性
- 情报有精度层级
- 每个参战者在每个时间点的情报清单应可追溯

### 7.3 死亡的叙事重量

- 每个参战者的死亡必须在至少一个其他角色的视角中产生反响
- 如果一人被消灭后其余人的行为和策略没有任何变化——设计有问题
- 参战者死亡意味着英灵意识的完全消散，这个不可逆性本身就是叙事重量的来源

### 7.4 非战斗场景要求

- 每两场战斗之间至少有一个完整的非战斗段落（不是过渡句，而是有信息价值的场景）
- 非战斗场景必须至少完成以下功能之一：揭露情报 / 建立或破坏关系 / 展示角色日常 / 推进角色内心弧线
- 谈判场景中双方必须各有底牌和底线，禁止信息完全透明的坦诚对话

---

## 八、主角设定原则

### 8.1 主角的强度边界

- 主角可以很强，但必须有明确的、不可绕过的硬限制
- 弱点不能只是"性格缺陷"——必须是战斗层面的硬限制
- 强度成长必须有代价

### 8.2 主角不是唯一视角

- 主角的信息不可超过合理范围
- 主角的判断可以是错的
- 主角消灭其他参战者的过程不应每次都是压倒性胜利——至少一次战斗中主角应陷入真正的危险

### 8.3 主角与英灵力量的关系

主角身上承载的英灵力量与主角本人之间必须有独特的张力——这个张力是主角叙事弧线的核心之一。不可以是"英灵力量完全听话、毫无副作用"。
